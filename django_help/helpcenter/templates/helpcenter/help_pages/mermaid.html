Mermaid Diagrams & 3D Model

This page documents how Mermaid markdown is used to produce the semantic model and 3D visualizations in this repository.

Mermaid source

The canonical Mermaid markdown lives in the repository mermaid-diagram.md. It is converted into an intermediate model_meta.json which powers the viewers.

How to preview

  • Run the converter tooling to generate three_renderer/public/model_meta.json.
  • Open the 3D viewer at 3D Viewer (this embeds the running three_renderer viewer).

Developer notes

The conversion pipeline: mermaid-diagram.md → generate_model_json.js → model_meta.json → viewer. The viewer supports semantic, timeline and heatmap modes and can export glTF.
