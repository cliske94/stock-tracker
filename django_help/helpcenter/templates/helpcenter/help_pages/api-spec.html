API Specification

This document lists the main endpoints exposed by the backend:

  • POST /auth/register — body: { "username": "u", "password": "p" } returns { "token": "..." }
  • POST /auth/login — body: { "username": "u", "password": "p" } returns { "token": "..." }
  • GET /search?ticker=<sym> — requires Authorization: Bearer <token> — returns stock JSON
  • GET /stock?ticker=<sym> — same as search but includes HATEOAS links

Use the example curl commands in the top-level README for quick testing.
