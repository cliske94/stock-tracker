Application Suite Overview

This section provides quick links and descriptions for the deployed containers and services that make up the project.

  • three-renderer — Node/Express visualizer service.
  • cpp-stock-ui — C++ UI + sidecar deployment.
  • dashboard-service — Flask dashboard service.
  • python-watchlist-gui — Python GUI (noVNC wrapper).
  • prometheus — Prometheus metrics server.
  • grafana — Grafana dashboards.
  • loki — Loki log storage.
  • spring-backend — Spring Boot API backend.
  • angular-ui — Angular frontend.

Each container page contains a small Graphviz diagram describing the container image, ports, and role. These are rendered client-side using Viz.js.
