Using the Rust GUI

Build and run the Rust SDL2 GUI:

cd rust_hello_world
cargo build --release
BACKEND_URL=http://localhost:8080 ./target/release/rust_hello_world

Ensure SDL2 and SDL2-ttf system packages are installed. The GUI persists token to token.txt.
