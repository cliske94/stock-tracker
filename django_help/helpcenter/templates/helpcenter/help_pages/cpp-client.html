Using the C++ client

Many C++ examples live in the repo. Quick example:

cd cpp_stock_ui
make
./cpp_stock_ui

Or build with CMake at the repo root.

Notes & Troubleshooting

  • If you expose the C++ Qt UI with x11vnc/noVNC and see a black screen or connection issues, see the noVNC troubleshooting page for verification and restart steps.
