Visualizations Overview

This page documents the available visualizations and how they are generated.

  • Mermaid â†’ Semantic Model: The Mermaid markdown is parsed and converted into a semantic JSON model used across all visualizations.
  • 3D Viewer: A Three.js viewer loads a mesh (OBJ/glTF) and overlays semantic labels and timeline/heatmap layers.
  • Force Graph: D3 force-directed layout showing nodes and edges from the semantic model. Supports zoom/pan, hover tooltips, and click-to-highlight.
  • Treemap: Shows LOC-based area sizing, with tooltips and selectable nodes reporting repo, LOC, and import counts.

Developer workflow:

  1. Edit mermaid-diagram.md in the repository.
  2. Run the converter: node three_renderer/tools/generate_model_json.js and optionally node three_renderer/tools/analyze_deps.js and node three_renderer/tools/generate_git_history.js.
  3. Sync to the running viewer: ./three_renderer/tools/post_model_meta_dev.sh or use the port-forwarded embed to reload the viewer.

See the embedded pages for interactive exploration: 3D Viewer, Force Graph, Treemap.
