3D Viewer (GLTF)

This page embeds the Three.js GLTF viewer from the three_renderer service (must be running and port-forwarded locally).

If you run the development port-forward helper, the viewer is available at http://localhost:9095/viewer_gltf.html.

Use the mode selector inside the embedded viewer to switch between semantic, timeline, and heatmap modes.
